Welcome to ADA Slope Compliance Tool

Analyze pedestrian pathways for Americans with Disabilities Act (ADA) compliance

How It Works

1. Upload Your Data

Provide a Digital Elevation Model (DEM) file showing terrain elevations. Optionally include pedestrian path data, or let the tool fetch paths from OpenStreetMap automatically.

2. Automatic Analysis

The tool calculates slope percentages along each path segment and checks them against ADA standards: ≤5% for running slope, ≤2.083% for cross slope.

3. Interactive Results

View results on an interactive map with color-coded paths (green = compliant, red = non-compliant) and detailed statistics for each segment.

Requirements

  • DEM File: Must be in projected coordinate system (UTM, State Plane, etc.) - not geographic (lat/lon)
  • File Format: GeoTIFF (.tif or .tiff files only)
  • Coordinate Units: DEM must use meters, not degrees
  • Path Data: Optional GeoJSON file, or automatic fetch from OpenStreetMap

Need Help?

If your DEM is in geographic coordinates (EPSG:4326), reproject it using QGIS or similar GIS software to a projected coordinate system like UTM before uploading.
